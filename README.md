川海商城，使用yaf开发

1.数据库信息修改：protected\config\application.ini

2.环境要求：php>=5.4  yaf最新版即可,开启yaf命名空间

3.微信支付证书位置：\m\weixin\cert\

4.shell服务：php -f /web路径/cmd.php request_uri=/cmd/order/do
